# Architecting for the Web on AWS

This session provides a practical guide to designing and deploying modern web architectures on Amazon Web Services, along with an open-source reference implementation to support real-world deployments. Attendees will learn how to evolve a codebase into a fully operational, production-grade application running on AWS, with a strong emphasis on well-architected, reliable, and cost-efficient systems.

We will explore the architectural decisions behind web application delivery, including how to structure microservices, leverage managed AWS compute, and implement robust systems. We will also discuss global content delivery strategies, including edge computing, security best practices, and additional patterns for building resilient cloud-native systems.

Designed for developers, cloud engineers, and architects, this talk will equip you with actionable insights and a reference AWS implementation that can accelerate your own projects. Whether you are launching your first application on AWS or refining an existing architecture, you will leave with clear patterns and reusable infrastructure ideas grounded in a real, open-source system.
